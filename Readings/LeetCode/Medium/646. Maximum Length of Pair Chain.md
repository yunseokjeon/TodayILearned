646. Maximum Length of Pair Chain

https://leetcode.com/problems/maximum-length-of-pair-chain/

```Java
class Solution {
    public int findLongestChain(int[][] pairs) {
        Arrays.sort(pairs, (a, b) -> a[1] - b[1]);
        int result = 0, current = Integer.MIN_VALUE;
        for (int[] pair : pairs) {
            if (current < pair[0]) {
                current = pair[1];
                result++;
            }
        }
        return result;
    }
}
```
